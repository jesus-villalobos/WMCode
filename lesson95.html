Lesson Content:

Before continuing, although we won’t use them right away, it is good to note that there are additional methods for us to use on lists. On a list, lst, they are:
  • lst.remove(*item*) - Removes *item* from the list if it exists
  • lst.reverse() - Reverses the order of the list
  • lst.index(*item*) - Returns the index of *item* in the list

Real Application of Lists

Suppose we are programming an application that hosts a large number of students. We want to greet them all, so our first task is to write a method that will take care of this for us.

How can we do this now?

Greeter -- Simple

See the recording for a better walk-through!

As a beginning, the easiest way to accomplish this task is to iterate through the given list with the needed information, and extract it using indexing. We can make use of our while loop to make this lookup and extraction extra simple.

This will look something like:

def greeter(lst):
	counter = 0
	while counter < len(lst):
		print("Hello " + lst[counter] + "!")
		counter += 1

New Situation

Now that we solved the greeter problem, our product manager approaches us with a different problem. They say:

“There was an error in the names where the first third of every list contained fake names. Modify your greeter method to only greet the second and third thirds of every list.”

Let's go through different approaches.

Approach 1 -- Building on Indexing

See the video for a better walk-through! We can simply build off what we already did above, and add a condition. It will look something like: 
def greeter(lst):
	counter = 0
	while counter < len(lst):
		if counter > int(len(lst) / 3):
			print("Hello " + lst[counter] + "!")
		counter += 1

Approach 2 -- List Slicing

While the last approach definitely works, there is a much more efficient way to get only the information that we need from the given list. We are going to use a new concept known as list slicing. Quite literally how it sounds, we are going to cut the input list down to the second and third thirds (the last 2 parts) of the list.

List slicing notation is:

*LIST*[*start_index* : *end_index*]
There are 3 variations to using a slice that are typically used. They are: 
*LIST*[: *end_index*]       → “from the beginning to *end_index*”

*LIST*[*start_index* :]     → “from *start_index* to the end”

*LIST*[ : ]                 → “from the beginning to the end”
We can put this together with our list accessing using a while loop, and come up with a function like: 
def greeter(lst):
    counter = 0
    slice = len(lst) // 3
    new_lst = lst[slice:]
    while counter < len(new_lst):
		print("Hello " + new_lst[counter] + "!")
		counter += 1

One More Job

Now, suppose our product manager approaches us with a different problem:

“Great job with fixing greeter! Now, we have decided that our users need to be greeted differently depending on when they joined our platform:

  • If they joined at a multiple of 5, greet them saying, "You are a chosen one, ___"
  • If they joined at an even position, greet them saying, "Welcome in, ___!"
  • If they joined at an odd position, greet them saying, "Why hello there, ___ :)"

Now, we can put together what we've learned with if-elif-else blocks and come up with a somewhat long solution, but, in my opinion, the easiest one. It would be something like:

def greeter(lst):
    slice = int(len(lst) / 3)
	new_lst = lst[slice:]
	counter = 0
	while counter < len(new_lst):
    	if counter % 5 == 0:
        	print("You are a chosen one, " + new_lst[counter])
    	elif counter % 2 == 0:
        	print("Welcome in, " + new_lst[counter] + "!")
    	else:
        	print("Why hello there, " + new_lst[counter] + " :)")
    	counter += 1
