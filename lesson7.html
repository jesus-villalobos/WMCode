Lesson Content:

The while Loop - Motivation

In order to effectively introduce the idea of what the while loop does, let's look at a real- life example of where a loop is used. The example that we are going to use is:
Students getting lunch at lunch-time.

Demonstration

To visualize this situation, we can think of any arbitrary line of students. The line can have any number of students in it, and during the lunch period, all of them must recieve lunch. The best way to ensure that everyone gets their lunch is:
  1. Serve lunch to the first student in line
  2. Continue serving lunch to the students behind them until there is no one left
This simple process will work for ANY number of students.
See the slides for a more involved, visual demonstration, or the video for a walk-through of the problem by Jesus.

Analysis and Intuition

Looking at a loop like this, we recognize that it is a fool-proof method of making sure that all students get their lunch. While there are still students in line, whoever the lunch-person is will continue serving lunch. This loop (serving lunch) will only end once there are no students left in the line.
If we think about this intuitively, we still go through the exact same steps that we went through in the section above, just a little more clearly:
  1. We start off with a set number of students
  2. While there are still students left in the line, we will serve lunch
  3. Once there is no one left in the line, we will stop serving lunch
This intuitive definition is exactly what we need to write implement this problem into Python code.

The while Loop

The while loop is a special unit of Python's control-flow syntac that allows us to do something (purposefully vague) while a condition is True.
In this case, we will serve lunch while there are students to serve.

The Initial Condition

Quite intuitively, we need some condition that begins as True and eventually becomes False.
More often than not, we are going to make use of the Python operators 
<      <=       >       >=      !=
to compare an end-goal with some sort of a count.
For the example of the lunch line, this can look something like: 
students_in_line = 25
while students_in_line > 0

The Body

After we have our initial condition set up, we need to give our while loop its own body. This is where we’ll put anything that we want to happen in every iteration. 
students_in_line = 25
while students_in_line > 0:
    print(“One more student got lunch!”)

The End-Condition

It is VERY important to make sure that our loop ends eventually, we need to make sure that our condition eventually becomes False. To do this, we need to decrease the number of students_in_line after every step in our loop. That will look something like: 
students_in_line = 25
while students_in_line > 0:
    print(“One more student got lunch!”)
    students_in_line = students_in_line - 1

Refactoring Loop to Function

Up until this point, our loop stands on its own. With a few minor changes, we can alter our loop into a fully user-enabled function.

Step 1: Function Name and Parameter

By now, we know that function names are important in making sure that our code is readable and understandable. So, quite intuitively, a good name for this function would be serve_lunch. To make this new function ready to serve lunch to ANY number of students, we can pass in the number of students_in_line as a parameter to our function. This will look like: 
def serve_lunch(students_in_line)

Step 2: Docstring

We can simply, include all of the things that we have talked about, previously, as our documentation for this function. The name of the function is already quite telling of what it will do, but documentation is never bad. It will look something like: 
def serve_lunch(students_in_line):
    """serve_lunch will serve lunch to however many students_in_line there are."""

Step 3: Function Body

The final step is to move the loop down from the previous sections. So, putting everything together, and adding one final closing statement, our final function is: 
def serve_lunch(students_in_line):
    """serve_lunch will serve lunch to however many students_in_line there are."""
    while students_in_line > 0:
        print(“One more student got lunch!”)
	    students_in_line = students_in_line - 1
    print(“All students got lunch!”)

A More Involved Examlple

In this example, we will tie together all of the concepts that we have covered in these lessons thus far. This example will include:
  • Variables and Assignment
  • Operators
  • Functions
  • Boolean Operators
  • The While Loop
You can watch the full walk-through in the video. There, there is a full walk-through of the function. The end product is: 
def make_even(x):
    """Iterate through values from 0 to x. Even numbers
    are going to be printed. Odd numbers are going to be multiplied by 3 and have 1 added to them.

    >>> make_even(10)
    0
    4
    2
    10
    4
    16
    6
    22
    8
    28
    10
    """
    current_value = 0
    while current_value <= x:
        if current_value % 2 == 1:
            print(current_value * 3 + 1)
        else:
            print(current_value)

        current_value = current_value + 1
